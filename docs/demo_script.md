# Demo Script

## Problem

BMS retrofit projects require engineers to manually review specs, cut sheets, submittals, and webpages, then convert findings into RFIs, tasks, points lists, and review notes.

## Solution

SpecFlow AI converts documents into clean Markdown and extracts workflow-ready engineering outputs.

## Demo Flow

1. Upload a PDF or paste a URL.
2. Generate Markdown.
3. Extract RFIs, tasks, equipment, risks, and points.
4. Export CSV or Markdown report.

## Value

- Reduces first-pass document review time
- Helps junior staff ramp faster
- Standardizes document review outputs
- Creates reusable engineering workflow artifacts
